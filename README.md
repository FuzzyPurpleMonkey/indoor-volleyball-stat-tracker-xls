# indoor-volleyball-stat-tracker-xls
A simple stat tracking spreadsheet for indoor volleyball to help your team improve their play.

# Instructions
1. Update the player data on the Legend sheet. 
- Each Player Key and Player Number must be unique.
- 'tt/88', 'xx/0' and 'p/99' are reserved.
2. During the game, enter in data into the Data sheet in the Raw Data column. See example.xlsx for an example of the codes.

Hints/tips:
- Entering in data during the game is FRANTIC. Don't stress if you mess something up, just keep entering data and fix it up later. 
- It helps if one person does serves and kills, and the other does everything else.
